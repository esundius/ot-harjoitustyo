# Ohjelmistotekniikka, harjoitustyö

## Mitä olen tekemässä?

Suunnitteilla **elokuvatietokanta**, johon voi lisätä (elokuvien lisäksi) elokuvia *watchlistaan* ja *katsottujen elokuvien listaan*. Kiinnostavaa olisi hakea elokuvien metadataa netistä (esim. *The Movie Database*:sta (TMDB)).

*Mahdollisesti* tuon elokuva-ohjelman sijaan **kirjakokoelma-ohjelma**, en tiedä. Oikeastaan olen miettinyt, että ohjelmaa voisi laajentaa sekä elokuville, kirjoille että musikkilevyille. Mutta se on sitten tulevaisuuden homma...
